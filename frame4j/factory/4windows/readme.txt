This does not works on 64 bit Windows and Java. On 32 Bit
Put the .jar in  %JAVA_HOME%\jre\lib\ext\
the .properties in %JAVA_HOME%\jre\lib\
and the .dll in  %JAVA_HOME%\jre\bin\

For 64 bit Windows use the files provided in directory ..\4win64\ 

Test it with Frame4J's ShowPorts by just
java.exe ShowPorts
