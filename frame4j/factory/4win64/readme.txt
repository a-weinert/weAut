This works on 64 bit Windows and Java only.
Put the .jar in  %JAVA_HOME%\jre\lib\ext 
and the .dll in  %JAVA_HOME%\jre\bin

For 32 bit Windows use the files provided in directory ..\4windows\ 

Test it with Frame4J's ShowPorts by just
java.exe ShowPorts
